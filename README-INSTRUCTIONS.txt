MOONSTRUCK CULTIVATION — SITE FILES

このフォルダには以下の完成版ファイルが入っています。

- index.html
- style.css
- about.html
- archive.html
- details.html

使い方:
1. ZIPを解凍します。
2. 5ファイルを、Macのデスクトップにある
   moonstruckcultivation.github.io フォルダへ入れます。
3. 「同じ名前のファイルがあります」と出た場合は「置き換える」を選びます。
4. 既存の logo.png と favicon.png はそのまま残してください。
5. GitHub Desktopで Summary に
   Complete site version 1.0
   と入力します。
6. Commit to main → Push origin の順に押します。

重要:
index.html 内のInstagramリンクは、現在
https://www.instagram.com/
になっています。実際のアカウントURLが決まったら変更してください。
